7.2 MODEL PIPELINE:
------------------
Input Abstract → 
  ├── General BERT Encoding
  ├── Scientific BERT Encoding  
  ├── Enhanced TF-IDF Vectorization
  ├── Title Processing
  └── Keyword Extraction
     ↓
Similarity Computation →
  ├── Cosine Similarity (each model)
  ├── Impact Factor Boost
  └── Field Matching Boost
     ↓
Weighted Score Combination → Final Rankings

7.3 SCORING FORMULA:
-------------------
Final Score = (
    0.25 * general_bert_similarity +
    0.25 * scientific_bert_similarity +
    0.20 * tfidf_similarity +
    0.10 * title_similarity +
    0.10 * keyword_similarity +
    0.05 * impact_factor_boost +
    0.05 * field_matching_boost
)
